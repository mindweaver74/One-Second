# 1초를 눌러봐!

> 정확히 1.000초에 손을 떼세요. 오차가 적을수록 높은 점수!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/One-Second](https://mindweaver74.github.io/One-Second)

---

## 게임 방법

1. 버튼을 **탭하면** 타이머가 시작됩니다
2. 1.000초라고 느껴질 때 **다시 탭하세요**
3. 오차가 적을수록 높은 등급과 점수!

---

## 등급 체계

| 오차 | 등급 | 점수 |
|------|------|------|
| 0.010초 이내 | 신의 손 | 1000점 |
| 0.030초 이내 | 달인 | 800점 |
| 0.060초 이내 | 고수 | 600점 |
| 0.100초 이내 | 보통 | 400점 |
| 0.200초 이내 | 아쉽다 | 200점 |
| 그 이상 | 다시하자 | 50점 |

---

## 특징

- 타이머가 1초 근접 시 링이 황금색으로 변하며 시각적 힌트 제공
- 매일 새로운 데일리 챌린지 (신의 손 N회, 달인 N회 등)
- 결과 공유 버튼 (카카오/인스타/클립보드)
- 연속 도전 스트릭 기록
- 최고 기록 / 평균 오차 / 총 도전 횟수 저장

---

## 긴장감 효과음

버튼을 누르는 순간부터 1초를 향해 음이 점점 높아지고 떨림이 강해지다가,
1초가 지나면 빠르게 식어가는 긴장감 있는 효과음이 재생됩니다.

| 구간 | 효과음 |
|------|--------|
| 0초 시작 | 낮은 음이 조용히 등장 |
| 0 → 1초 | 음이 점점 높아지고 떨림 가속 |
| 딱 1초 | 최고조 |
| 1초 초과 | 빠르게 식어감 |

결과 등급별로도 다른 효과음이 재생됩니다.

| 등급 | 효과음 |
|------|--------|
| 신의 손 | 5음 상승 팡파레 |
| 달인 | 밝은 두 음 |
| 고수 | 맑은 단음 |
| 보통 | 낮은 단음 |
| 다시하자 | 하강 실망음 |
